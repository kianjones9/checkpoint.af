# Readme

## Introduction
Checkpoint is a convenient uitilty for saving, versioning, and migrating snapshots of a Letta agent's state as an [agentfile](https://github.com/letta-ai/agent-file).

## Functionality

### save

Save a snapshot of the agent

### rollback (unimplemented)
Return the agent to a previously snapshotted state (optionally in place)

### migrate (unimplemented)
Migrate a "rolled-back" agent to a different snapshotted state in it's future
